# پروژه حل مسئله 8 وزیر با الگوریتم ژنتیک

این پروژه برای تمرین مفاهیم Git/GitHub و همچنین پیاده‌سازی الگوریتم ژنتیک جهت حل مسئله n-Queen طراحی شده است.

## ساختار پروژه
- `board.py`: کلاس Board شامل موقعیت وزیرها و تابع fitness
- `algorithm.py`: شامل توابع generate_population، selection، crossover، mutate و genetic_algorithm
- `main.py`: اجرای الگوریتم و نمایش نتیجه
- `description.txt`: ثبت تغییرات گام‌به‌گام

## نحوه اجرا
```bash
python main.py
